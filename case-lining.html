📋 项目背景

李宁集团是中国领先的运动品牌，在全球拥有庞大的业务网络。一期项目成功上线后，李宁的合同管理已基本实现线上化流转，但在履约管理环节仍存在断点，成为制约合同管理效率提升的关键瓶颈。

李宁二期项目是在一期基础上的深化升级，聚焦履约管理这一核心痛点，同时开展总结摘要、合同问答等智能化功能。该项目是与李宁保持每年持续深耕合作的重要组成部分，主要针对重大合同类型进行履约管理，包括金额100万以上、涉及收付款、赞助联名代言及店铺相关合同。

✨ 标杆客户持续深耕 ✨ 智能履约管理 ✨ 轻咨询式售前

🎯 客户核心需求

1. 履约管理断点打通

一期项目后，合同除履约外已在系统流转，但履约环节仍依赖线下管理，导致：

  • 履约节点跟踪困难，容易遗漏关键时间点
  • 履约责任不明确，多部门协同效率低
  • 履约数据分散，无法形成统一视图
  • 合同变更后履约信息同步滞后，容易产生执行偏差

2. 重大合同精细化管理

针对不同类型的重大合同，需要差异化的履约管理方案：

  • 租赁联营类合同：需跟踪开业日期、装修完成时间等关键节点
  • 营销类合同：需管控形象使用、宣传节点等义务项，避免权益浪费
  • 支出类合同：需识别付款条件、验收标准，避免财务执行偏差
  • 收入类合同：需跟踪收款节点和条件，确保及时回款

3. AI 智能化能力提升

希望通过 AI 技术大幅提升履约管理效率：

  • 智能提取履约条款，减少手动录入工作量
  • 自动生成履约计划，提高创建准确性
  • 智能预警和提醒，确保节点不遗漏
  • 合同摘要自动生成，提升协同效率

4. 系统数据闭环管理

实现合同全生命周期数据闭环：

  • 与 OA 系统数据同步，打通审批流程
  • 与财务系统对接，构建付款凭据链条
  • 履约状态可追溯、可复核，确保合规
  • 合同变更自动触发履约更新，保持数据一致性

💡 MeFlow Agent 解决方案

智能履约管理核心流程

幂律智能为李宁设计了完整的智能履约管理流程：

  1. 上传合同文本 → 支持多种合同类型和格式
  2. 大模型自动提取履约条款 → 结构化识别关键要素
  3. 自动生成履约计划 → 智能规划履约任务和节点
  4. 自动化智能预警 → 提前提醒关键节点和风险
  5. 自动跟踪 → 实时同步履约进展
  6. 闭环管理 → 完整记录履约全过程

核心功能模块

  1. 履约分类管理
    支持自定义配置各类履约类型和履约任务表单，实现各类履约任务精细化管理。针对租赁联营、营销、支出、收入等不同类型合同，提供差异化的履约管理方案。
  2. 履约任务创建
    结合企业管理习惯，支持在合同起草、审批、签署等多阶段创建履约任务，实现精细化任务管理。用户可根据业务需要灵活选择创建时机。
  3. AI 智能辅助创建履约任务
    AI 智能解析合同履行关键要素，基于合同及附件，按照履约任务成组进行抓取，支持配置每一组履约任务的 prompt。智能生成履约任务，辅助用户快速创建。
  4. 履约任务执行管理
    管控全局合同履行进度，让用户了解合同的履约最新进展。当履行即将延期或签方发生工商异动时，自动进行预警，确保风险可控。
  5. 营销权益统计表智能提取
    通过大模型，根据【合同履约跟踪表】智能提取合同履约关键信息，形成线上履约跟踪表，实现营销权益的统一管理。
  6. 财务统计表智能提取和推送
    支持自定义配置合同表单或财务统计表单，并支持自定义配置每个字段、每张表的提取 prompt，进行按表抓取。与财务系统对接，自动推送数据。

各业务场景精细化方案

  • 租赁联营类合同：智能结构化抽取合同重点条款，统一管控口径；跟踪"开业日期"、"装修完成时间"等关键节点；降低因合同执行落差引发的门店运营纠纷
  • 营销类合同：合同义务结构化后，明确"形象使用"、"宣传节点"等义务项；履约节点系统提醒，避免时间冲突/失期；降低因遗漏、模糊条款造成的品牌风险和权益浪费
  • 支出类合同：自动结构化识别付款条件、验收标准，避免财务执行偏差；履约状态同步财务系统，构建清晰付款凭据链条；减少因履约凭证缺失引发的付款风险与内控漏洞
  • 收入类合同：智能提取收款条件和节点，确保及时跟进；自动提醒收款时间，避免回款延迟

🔑 关键成功因素

1. 标杆客户持续深耕

李宁作为标杆客户，二期项目具有重要的示范意义。通过一期项目建立的良好合作关系，为二期项目的顺利推进奠定了坚实基础。每年持续深耕合作的策略，确保了客户关系的稳定和项目的可持续性。

2. 轻咨询式售前工作

陈莉娟作为售前支持核心成员，采用轻咨询性质的售前方法，从客户视角调研了解现状全貌，梳理问题与难点、项目价值点，并给出建设规划建议，思考更全面。这种深度的业务理解确保了方案的针对性和落地性。

3. 全面的需求梳理

团队协助李宁全面梳理各类合同的履约现状、难点及是否需要做履约。具体需要通过系统管控哪些需求事项，并针对预期管控事项，分场景设计方案。最终整理形成整体方案，进行后续立项汇报。

4. 坚持合理价格

面对客户多次"威胁"（如预算超太多就没法合作或要自研），团队顶住压力，坚持了合理的价格，同时让客户认识到履约管理的价值。最终通过调整合作方式（签框架合同），既解决了客户预算问题，又保证了项目质量。

5. 创新的合作模式

项目整体预算 58.8 万，采用框架合同的方式：今年有 30 万预算，先签 58.8 万框架项目，今年下 30 万订单并框定内容，明年1月1号下剩余订单。付款方式为每下订单付 50% 预付款，验收付 50%。这种灵活的合作模式解决了双方的实际问题。

6. 价值提炼与高层汇报

团队对履约价值进行深度提炼，并向李宁决策层领导汇报，让领导认识到项目的价值。这种高层沟通确保了项目得到足够的重视和资源支持。

💡 经验总结与复用价值

可复用的成功经验

  1. 标杆客户深耕策略：通过持续深耕标杆客户，建立长期合作关系，每年推进新的项目，实现客户价值最大化
  2. 轻咨询式售前方法：从客户视角调研梳理，提供建设规划建议，这种方法可复用到其他复杂项目的售前工作
  3. 精细化业务场景方案：针对不同业务场景（租赁联营、营销、支出、收入）设计差异化方案，可应用到其他行业客户
  4. 灵活的合作模式：框架合同+分批下单的方式，既解决客户预算问题，又保证项目质量，可作为其他项目的参考
  5. 价值量化与提炼：通过详细的价值量化分析，让客户认识到项目价值，为高层决策提供有力支撑

标杆客户的示范效应

  • 走进李宁活动：借助标杆客户发声，帮助推动其他项目合作，新增成交了质信智购、中英人寿、字节跳动等客户
  • 二期合作推广：晶澳、颐和嘉业等客户也在规划二期合作，李宁案例成为有力背书
  • 履约管理标杆：李宁二期的智能履约管理方案，可作为其他运动品牌、零售企业的参考案例

行业拓展机会

  • 运动品牌：安踏、特步、361度等运动品牌同样面临履约管理挑战，李宁案例可快速复制
  • 零售企业：有大量店铺租赁、营销合同的零售企业，可应用相同的履约管理方案
  • 制造企业：有大量采购、销售合同的制造企业，可应用支出类、收入类履约管理方案

待优化的方面

  • 项目前期预算差距较大，导致项目启动周期较长，后续应加强成本测算和客户预期管理
  • 智能提取的准确率仍需在实际应用中不断优化，根据客户反馈持续改进模型
  • 与 OA、财务等系统的集成需要提前规划，确保数据流转顺畅
